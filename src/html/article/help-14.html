Web前端开发工程师Leader/ Web前端开发工程师
岗位职责：
1、负责网站的页面制作、前台效果的实现；
2、负责团队项目和产品的前后端交互的开发；
3、通过技术提升用户体验和可用性； 		岗位要求：
1、熟悉W3C标准；有响应式网站设计经验；能处理IE8以上浏览器兼容性问题；
2、使用过一种以上的前端框架，如 Bootstrap、Foundation；有Sass、LESS等使用经验；
3、掌握 jQuery / AngularJS / React 等JavaScript框架/类库；
4、熟练使用Git进行代码管理；
5、有独立完成前端模块设计与实现的经验；

优先录用：
参与过开源项目，有Github或技术博客展示自己以往技术沉淀者优先；投递简历时，
请随简历附上代表作品（图像、文档或成品链接等）。
PHP开发工程师
岗位职责：
1、负责互联网产品的后端开发；
2、根据设计要求独立完成代码编写及代码维护工作；
3、配合设计、前端、产品等人员完成项目。 		岗位要求：
1、熟悉 PHP，至少熟悉一种主流PHP框架，有较强的架构能力和良好代码规范；
2、了解 Redis、Memcached 等 NoSQL 技术；
3、了解 Linux 服务器操作系统；
4、对 MySQL 数据库有较为深刻的理解；
5、热衷于提高系统性能和可扩展性；
6、熟悉基本软件开发流程和工具，例如单元测试、版本管理、Bug管理等；
7、熟练使用Git进行代码管理；
8、逻辑思维能力强，做事有条理，具备较强的分析问题和解决问题的能力；
9、英语水平过硬，阅读英文技术书籍文章无压力；
测试工程师
岗位职责：
1、承担项目测试,包括测试需求分析,计划方案制定,测试用例设计和执行,完成测试报告；
2、根据产品功能说明编写测试计划，设计测试用例和测试数据；
3、执行具体测试任务并确认测试结果，完成测试报告以及测试结果分析；
4、在测试各个环节过程中主动与开发人员沟通，保证测试工作的有效性和完整性；
5、搭建测试环境，管理测试代码，进行测试代码的部署与调试；
6、能够快速地反映问题，描述问题，能对项目提出合理化建议；
7、良好的沟通,积极协助和驱动问题解决,高效的执行流程,保证项目质量和进度的达成；
8、保障线上质量，能主动承担责任，抗压能力强。 		岗位要求：
1、1年以上WEB项目测试经验，可独立完成自动化测试，熟练白盒测试；
2、精通测试理论、测试策略、测试用例设计方法等相关测试技术；
3、熟悉Linux或Unix操作系统；
4、熟悉My SQL等常用数据库的操作；
5、至少熟悉一门开发语言和很好的编程能力，JavaScript/PHP/Python；
6、能够独立带领团队完成工作者优先考虑；有很强的责任心，做事严谨，良好的书面表达和工作总结能力；
7、较强的沟通、协调能力、分析问题和解决问题能力；
8、有电商工作背景优先考虑，有开发经验者优先考虑。
采购专员/主管
岗位职责：
1、熟悉家具，家居家饰行业的款式，材料及工艺流程，有成本分析能力。
2、负责新产品开发，确保产品开发的准时率及成功率。
3、供应商开发及采购管理，确保有竞争力的采购价格及订单跟进。
4、有效的控制供应商产品质量，预防及改善质量问题，降低市场投诉率并达成公司相关绩效要求。
5、供应商的交货期管理，确保供应商出货准时率。 		岗位要求：
1、国家统招大专以上学历，理工或管理类专业。
2、三年以上家具采购，产品开发或质量管理工作经验。具有IKEA（宜家）、HOMEDEPOT(家得宝)、KINGFISHER翠丰(百安居) 等外资公司或者国内大型家居企业，家居家装电商背景优先；
3、有家具，家居用品或家纺供应商资源及开发能力。条件稍逊者可应聘采购专员。
4、熟练操作ERP,办公软件及PPT，基本的英文读写能力。
主编
工作职责：
1、确定产品内容整体运营方向以及目标；
2、规划资讯板块的栏目及内容，与产品部跟踪商品详情页的布局；
3、能针对商品描述或者文章描述进行优化排版，善于总结规律；
4、提升产品的媒体属性，扩大业内内容品牌影响力； 		岗位要求：
1、熟悉家居行业，熟悉家居方向内容运营；
2、3年以上互联网内容运营经验，家居方向经验者优先；
3、关注业内发展动态，并对行业发展有一定前瞻性 ；
4、良好的文字驾驭能力，能够撰写相关内容文章；
5、具有出色的管理能力，具备带领编辑团队达成工作目标的能力。
